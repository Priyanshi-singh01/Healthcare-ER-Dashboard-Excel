# Healthcare-ER-Dashboard-Exce
## Objective
To analyze emergency room patient data and identify key
patterns related to patient flow, waiting time, and satisfaction.
## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts
- Slicers

## Key KPIs
- Total Patients
- Average Waiting Time
- Patient Satisfaction Score

## Insights
- Adult patients form the largest group.
- Certain departments receive higher ER traffic.
- Waiting time directly impacts satisfaction.
